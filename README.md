# Magic Calculator

2026马年春晚计算器魔术纯享速通小工具，无需安装，1分钟速通马年春晚，祝大家马年行大运，马到成功！！！

## 功能特性

- iOS 风格计算器布局（数字、运算符、百分比、正负号、退格）
- 基础运算：`+`、`−`、`×`、`÷`
- 自适应显示字号与千分位格式化
- 移动端交互优化（防误触、禁双击缩放、全屏沉浸）
- 🎩 魔术模式：通过隐藏触发区域启动“目标时间”预测演示
- 🔧 隐藏配置模式：支持调整魔术时间缓冲秒数（0~60）


## 在线运行

这是一个静态页面项目，可直接打开[应用网址](https://magicalc.myhloli.com/)使用。

## iOS 添加到桌面

在 iOS 设备上快速访问此应用：

1. 使用 Safari 浏览器打开[应用网址](https://magicalc.myhloli.com/)
2. 点击底部工具栏的 **分享按钮**（向上箭头）
3. 向下滑动，选择 **"添加到主屏幕"**
4. 输入应用名称（默认为"Calc"）
5. 点击 **"添加"** 确认

添加后，应用将在 iOS 主屏幕上显示为独立图标，无需打开浏览器即可快速启动。

## 使用说明

### 普通计算模式

- 点击数字输入
- 点击运算符继续链式计算
- `AC/C` 清空
- `⌫` 退格
- `%` 百分比
- `+/−` 切换正负

### 🎩 魔术模式（隐藏）

大致流程：

1. 先找几个随机观众（非托）提供几个随机数，可以使用+-×÷计算，但是需要保证计算结果是个小于8位数的整数，10月份之前需要是小于7位数，为保证安全，请尽量只使用+法运算。
2. 先进行正常输入并按下 `+`（进入“准备状态”）
3. 点击显示区域左侧的透明“秘密触发区”，并将手机屏幕朝下，找几个前排观众点击屏幕。
3. 每点击屏幕一次，会逐位显示一个“修正数字”，过程中可以给观众展示1~2次输入的“随机数字”，需要注意点击的时候需要屏幕朝下。
4. 修正数字大概7~8位，正常2~3个观众点击即可，点击完成后向所有观众展示当前输入的一长串“随机数字”，并要求其他观众输入相同数字。
5. 最后按 `=`，结果会对齐到目标时间格式：`MMDDhhmm`

说明：目标时间会根据当前秒数和缓冲阈值，取“当前分钟”或“下一分钟”。

### 🔧 隐藏配置模式（缓冲秒数）

在清零状态（显示 `0`）下，输入隐藏口令：

`2026=====`

进入配置后：

1. 输入 `0~60`
2. 按 `=` 保存

该值用于控制时间目标计算的秒数缓冲阈值，即当前时间秒数大于等于阈值时，目标时间会配置为下一分钟，为最后一步随机数字展示和全体观众同步输入留足缓冲时间。


## 常见问题

### 1) 魔术结果和预期时间差一分钟

- 这是由缓冲秒数阈值控制的正常行为，可通过隐藏配置模式调整
