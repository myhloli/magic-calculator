# Magic Calculator

一个纯前端的“魔术计算器”单页应用，界面风格接近 iOS 计算器，支持基础四则运算，并内置“秘密触发”的时间魔术模式。

## 功能特性

- iOS 风格计算器布局（数字、运算符、百分比、正负号、退格）
- 基础运算：`+`、`−`、`×`、`÷`
- 自适应显示字号与千分位格式化
- 移动端交互优化（防误触、禁双击缩放、全屏沉浸）
- 🎩 魔术模式：通过隐藏触发区域启动“目标时间”预测演示
- 🔧 隐藏配置模式：支持调整魔术时间缓冲秒数（0~60）


## 本地运行

这是一个静态页面项目，可直接打开运行：

1. 双击 `index.html` 在浏览器中打开

## 使用说明

### 普通计算模式

- 点击数字输入
- 点击运算符继续链式计算
- `AC/C` 清空
- `⌫` 退格
- `%` 百分比
- `+/−` 切换正负

### 🎩 魔术模式（隐藏）

大致流程：

1. 先进行正常输入并按下 `+`（进入“准备状态”）
2. 点击显示区域左侧的透明“秘密触发区”
3. 后续点击屏幕会逐位显示一个“修正数字”
4. 最后按 `=`，结果会对齐到目标时间格式：`MMDDhhmm`

说明：目标时间会根据当前秒数和缓冲阈值，取“当前分钟”或“下一分钟”。

### 🔧 隐藏配置模式（缓冲秒数）

在清零状态（显示 `0`）下，输入隐藏口令：

`2026=====`

进入配置后：

1. 输入 `0~60`
2. 按 `=` 保存

该值用于控制时间目标计算的秒数缓冲阈值。


## 常见问题

### 1) 魔术结果和预期时间差一分钟

- 这是由缓冲秒数阈值控制的正常行为，可通过隐藏配置模式调整
